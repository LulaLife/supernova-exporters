### Claude Code Plugin exporter release notes

All updates to this exporter are documented in this file.

## 1.0.0 - 2026-06-16

### Added

- Export a Supernova context as a Claude Code plugin.
- Generate a plugin manifest, context-scoped Supernova MCP configuration, in-scope context skills, MCP usage guidance, optional feedback capture skill, optional marketplace manifest, and README.
- Use the workspace name as the fallback owner in generated Claude Code marketplace metadata.
- Use the configured repository URL in the generated README marketplace install command.
- Clarify the exporter value proposition and use sentence case in README titles.
